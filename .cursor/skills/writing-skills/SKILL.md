# Writing skills

## Instrucciones

1. Estructura: contexto → decisión o comportamiento → consecuencias o ejemplos.
2. Oraciones cortas; voz activa; términos consistentes en todo el doc.
3. Incluir el “por qué” cuando haya trade-offs o restricciones no obvias.
4. Revisar que comandos, rutas y nombres de variables coincidan con el código real.
