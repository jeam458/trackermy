# UI/UX pro max

## Instrucciones

1. Jerarquía visual clara: acción primaria evidente, secundarias discretas.
2. Estados explícitos: carga, vacío, error, éxito; nunca dejar al usuario a ciegas.
3. Copy conciso y orientado a la acción; evitar jerga innecesaria.
4. Coherencia con patrones ya usados en el producto (navegación, formularios, tablas).
