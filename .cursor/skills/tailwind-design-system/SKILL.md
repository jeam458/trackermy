# Tailwind design system

## Instrucciones

1. Reutilizar utilidades y patrones del proyecto antes de valores arbitrarios.
2. Mantener jerarquía tipográfica, espaciado y colores coherentes con el sistema.
3. Preferir componentes compuestos y variantes sobre clases duplicadas largas.
4. Verificar contraste, foco visible y tamaños táctiles en interacciones clave.
