# TypeScript — tipos avanzados

## Instrucciones

1. Preferir tipos derivados de datos reales (`satisfies`, `const` assertions) a copias manuales.
2. Usar unions discriminadas para estados mutuamente excluyentes.
3. Evitar `any`; usar `unknown` y acotar con type guards.
4. Extraer tipos reutilizables cuando un patrón se repite tres o más veces.
