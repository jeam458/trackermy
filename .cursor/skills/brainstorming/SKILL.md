# Brainstorming

## Cuándo aplicar

- Hay varias soluciones posibles y falta priorizar o acotar.
- Se necesita explorar riesgos, dependencias o alcance antes de codificar.

## Instrucciones

1. Reformular el objetivo en una frase clara y medible.
2. Proponer al menos tres enfoques distintos (incluyendo uno conservador y uno más ambicioso).
3. Listar pros, contras y supuestos de cada uno.
4. Recomendar un camino con criterios explícitos (tiempo, riesgo, mantenimiento).
